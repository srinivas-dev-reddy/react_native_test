# Setup

## Pre-requisites
```
node v16

# For Android 
Android Studio setup with emulator with 16gb internal storage

# For IOS
Xcode needed
```

## Installation
``` 
npm install
```

## Running
```
npm start

# For Android
npx react-native run-android

# For IOS
npx react-native run-ios
```
